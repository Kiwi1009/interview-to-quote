# 如何启动平台

## 🚀 快速启动（需要2个PowerShell窗口）

### 窗口1：启动后端

```powershell
cd backend
python run.py
```

**成功标志**：看到 `Uvicorn running on http://0.0.0.0:8000`

### 窗口2：启动前端

```powershell
cd frontend
npm run dev
```

**成功标志**：看到 `Local: http://localhost:5173/`

### 访问网页

在浏览器中打开：**http://localhost:5173/simple**

## ⚠️ 如果无法访问

### 检查1：服务是否在运行？

**检查后端**：
```powershell
# 应该能看到8000端口
netstat -ano | findstr :8000
```

**检查前端**：
```powershell
# 应该能看到5173端口
netstat -ano | findstr :5173
```

### 检查2：查看终端输出

- 后端终端：查看是否有错误信息
- 前端终端：查看是否有编译错误

### 检查3：手动启动

如果后台启动失败，请手动启动：

**后端**：
```powershell
cd C:\GITHUB_CODE\interview-to-quote\backend
python run.py
```

**前端**：
```powershell
cd C:\GITHUB_CODE\interview-to-quote\frontend
npm run dev
```

## 📝 使用步骤

1. 确保两个服务都在运行
2. 打开浏览器访问：http://localhost:5173/simple
3. 填写案件名称
4. 上传逐字稿文件
5. 点击"开始处理并生成文件"

---

## 🔧 常见问题

### 问题：端口被占用

**解决**：
- 关闭占用端口的程序
- 或修改端口配置

### 问题：npm install 失败

**解决**：
```powershell
cd frontend
npm cache clean --force
npm install
```

### 问题：Python模块缺失

**解决**：
```powershell
cd backend
pip install -r requirements.txt
```
